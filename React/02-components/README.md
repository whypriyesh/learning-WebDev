# 02 - React Components

In this project, I learned about React components and how to:
- Create reusable components
- Structure components effectively
- Pass data using props
- Build component hierarchies

## Key Components

- **Card Component** - A reusable card display component
- **App Component** - Main application component

## Concepts Covered

- Functional components
- JSX syntax
- Props and prop drilling
- Component composition
- Reusability and DRY principles

## Getting Started

```bash
npm install
npm run dev
```
