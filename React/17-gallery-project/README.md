# 17 - Gallery Project

Building a functional image gallery application:
- Displaying images in a grid
- Image selection and enlargement
- Navigation between images
- Filtering and sorting
- Responsive image layout
- API integration for images

## Getting Started

```bash
npm install
npm run dev
```

## Features Implemented

- Image grid display
- Modal/lightbox for full view
- Image navigation
- Image filtering
- Responsive design
- Loading states

## Concepts Applied

- useState for gallery state
- useEffect for data loading
- Component composition
- Event handling
- Conditional rendering
- CSS Grid/Flexbox layouts

## Technologies

- React hooks
- API integration
- CSS styling
- Responsive design patterns
