# 05 - CSS Styling in React

This project taught me different ways to style React components:
- Inline styles
- CSS modules
- Regular CSS files
- Scoped styling
- CSS best practices in React

## Getting Started

```bash
npm install
npm run dev
```

## Styling Approaches

- **Inline Styles** - JavaScript objects for styling
- **CSS Files** - Traditional CSS approach
- **CSS Modules** - Scoped CSS files
- **Class Names** - Dynamic class assignment

## What I Learned

- Pros and cons of each styling approach
- How to organize styles at scale
- Dynamic styling techniques
- CSS-in-JS alternatives
