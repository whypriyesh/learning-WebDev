# 13 - Notes App Project

A practical project building a functional notes application:
- Creating, reading, updating, and deleting notes
- Managing notes in state
- Form integration
- List rendering
- Local state management
- Real-world application structure

## Getting Started

```bash
npm install
npm run dev
```

## Features Implemented

- Add new notes
- Display notes list
- Edit existing notes
- Delete notes
- Simple UI for note management

## Concepts Applied

- useState for state management
- Component composition
- Event handling
- List rendering with keys
- Form handling
- CRUD operations
