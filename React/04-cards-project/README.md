# 04 - Cards Project

A practical project where I built a card-based UI system:
- Creating reusable card components
- Managing multiple cards in a list
- Styling cards consistently
- Building a complete layout

## Getting Started

```bash
npm install
npm run dev
```

## Project Features

- Multiple card variations
- Card component library
- Responsive design patterns
- Component composition examples

## Concepts Covered

- Component reusability
- Props for customization
- CSS styling in React
- Layout and spacing
