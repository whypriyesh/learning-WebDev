# 06 - Tailwind CSS in React

Learning Tailwind CSS and its integration with React:
- Tailwind utility classes
- Building components with Tailwind
- Responsive design with Tailwind
- Custom Tailwind configuration

## Getting Started

```bash
npm install
npm run dev
```

## Key Features

- Utility-first CSS approach
- Rapid component styling
- Responsive design patterns
- Theme customization
- Component libraries built with Tailwind

## Concepts Covered

- Tailwind setup in React
- Utility class combinations
- Responsive prefixes (sm, md, lg, xl)
- Dark mode support
- Custom configurations
