# 03 - Props Deep Dive

This project focuses on understanding props and how data flows through React components:
- Passing props to child components
- Props destructuring
- Default props
- Prop validation

## Getting Started

```bash
npm install
npm run dev
```

- How props enable component communication
- Props syntax and destructuring

- Data flow in React (top-down)
